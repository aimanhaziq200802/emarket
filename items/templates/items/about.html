About My Project

Project Objectives

The objective of this project is to develop an E-Market platform for small-scale groceries. This platform aims to connect local sellers with buyers, providing a convenient and efficient online marketplace for fresh produce and groceries.

Technologies Used

This project is built using the following technologies:

  • Django: A high-level Python web framework for rapid development and clean, pragmatic design.
  • Bootstrap: A front-end framework for developing responsive and mobile-first websites.
  • OpenWeatherMap API: Integrated for real-time location-based services.
  • SQLite: A lightweight database engine for storing project data.

My Story

This project was initiated as a part of my university coursework to address the growing need for an efficient and user-friendly online grocery shopping experience. I am a dedicated student with a passion for technology and a vision to create impactful solutions.

Throughout the development process, I have applied the best practices in software engineering, project management and more, ensuring a robust and scalable platform. I am proud of the progress I have made and am excited to share my work with the academic community and beyond.
